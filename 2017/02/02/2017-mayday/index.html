关于纯真

大二那年 准备买票 才知道有纯真论坛 后来注册了账号

去年 为了上海场 五吧和纯真同时盯着 后来 都团到了票 少数例子说明 五吧的位子比纯真更好

今天翻微博发现纯真发声明说有四位管理者退出了 其他事务会有他人接手

其实 我并不认识他们是谁 但是看到 一个因为热爱组织起来的非营利组织 已经存在14年 还是很感动

毕竟 暴民很强大 经历过团购的人都知道 从准备团购 到转账 填单子 每场数额很大 还会有人填错各种信息

而且一到要买票的时候论坛就会崩掉哈哈哈

所以 大家辛苦啦
